# AGENTS.md

This file provides guidance to coding agents when working with code in this repository.

## Live Site Ownership

- `https://michelstorms.com` is currently served by this `stormychel.github.io` repository via GitHub Pages.
- If a request is about changing the live Michel Storms website, make the change here.
- The separate `michelstorms.github.io` repository is not the live custom-domain source unless the Pages domain mapping is intentionally moved.

## Workflow Expectations

- For visual or content changes, make the change locally first and preview it before considering the task done.
- After editing, open the changed page from a local server in the browser so the result can be reviewed locally before deployment.
- Prefer using the local preview URL in the handoff when the task affects page layout, styling, or copy.
